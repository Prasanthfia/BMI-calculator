# ⚖️ BMI Calculator

A clean BMI calculator with both metric and imperial unit support.

## Features
- Switch between Metric (kg/cm) and Imperial (lb/in)
- Color-coded result: Underweight, Normal, Overweight, Obese
- Visual BMI scale bar
- Health tip based on result

## Built With
- HTML5
- CSS3
- Vanilla JavaScript

## How to Run
Clone the repo and open `index.html` in your browser.
